# Сборка и запуск прокси

## Конфигурация

- **Прокси** слушает порт **1802** (ENet)
- **RakSamp** сервер на **51.75.232.67:1801**
- **APK** настроен на подключение к **51.75.232.67:1802** (прокси)

Прокси должен работать на том же сервере, что и RakSamp (51.75.232.67).

## Сборка

```bash
cd /workspace/proxy
g++ -o enet_proxy enet_proxy.cpp -I./include libenet.a -lpthread
```

## Запуск

```bash
./enet_proxy 1802
```

Или с адресом RakSamp (для полного прокси в будущем):
```bash
./enet_proxy 1802  # слушает 1802, пересылает на 127.0.0.1:1801
```

## Тестирование

1. Запустите прокси на 51.75.232.67
2. Установите пропатченный APK
3. Введите ник, нажмите подключиться
4. В консоли прокси появятся пакеты от BR клиента

Текущая версия — логирование. Полный ENet↔RakNet перевод в разработке.
